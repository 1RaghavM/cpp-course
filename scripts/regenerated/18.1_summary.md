Sorting means arranging elements in order (ascending or descending). **Selection sort** is one of the simplest sorting algorithms to understand and implement. It works by repeatedly finding the smallest unsorted element and moving it to its correct position.

The algorithm divides the array into two regions: sorted (on the left) and unsorted (on the right). On each pass, it scans the unsorted region, finds the index of the smallest element, and swaps that element with the first unsorted element. After each pass, the sorted region grows by one.

For an array of size `n`, selection sort performs `n - 1` passes. On the first pass it examines all `n` elements, on the second pass `n - 1` elements, and so on. This gives roughly `n * (n - 1) / 2` comparisons total, making selection sort an **O(n²)** algorithm. It works fine for small arrays but becomes slow for large datasets.

Here is a complete example that sorts an array of integers in ascending order:

```cpp
#include <iostream>
#include <utility> // std::swap

int main()
{
    int arr[]{ 30, 50, 20, 10, 40 };
    int length{ 5 };

    for (int start{ 0 }; start < length - 1; ++start)
    {
        int minIndex{ start };
        for (int cur{ start + 1 }; cur < length; ++cur)
        {
            if (arr[cur] < arr[minIndex])
                minIndex = cur;
        }
        std::swap(arr[start], arr[minIndex]);
    }

    for (int i{ 0 }; i < length; ++i)
        std::cout << arr[i] << ' ';
}
```

Key points to remember:

- The outer loop tracks where the sorted region ends. It runs from index `0` to `length - 2` (the last element is automatically in place).
- The inner loop searches for the minimum value in the unsorted portion by comparing each element against the current minimum.
- `std::swap` (from `<utility>`) exchanges two values. You could write a manual swap with a temporary variable, but `std::swap` is cleaner.
- Selection sort is **not stable** — equal elements may change their relative order after sorting.
- This same pattern works with `std::array` or `std::vector`. Just replace the C-style array and use `.size()` for the length.
