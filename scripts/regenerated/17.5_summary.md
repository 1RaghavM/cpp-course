## Arrays of References via std::reference_wrapper

You might want to create a `std::array` of references — for example, to iterate over a set of existing variables and modify them in a loop. But `std::array<int&, 3>` won't compile. References are not objects: they have no address of their own, and containers like `std::array` require element types that are copyable and assignable. References satisfy neither requirement.

### The solution: std::reference_wrapper

`std::reference_wrapper<T>` (defined in `<functional>`) is a lightweight class that wraps a reference inside a copyable, assignable object. It behaves like a `T&` in most contexts but can be stored in arrays and other containers.

Use the helper functions `std::ref(x)` to create a `std::reference_wrapper<T>` and `std::cref(x)` to create a `std::reference_wrapper<const T>` (a read-only wrapper).

```cpp
#include <array>
#include <functional>
#include <iostream>

int main()
{
    int a{ 1 }, b{ 2 }, c{ 3 };
    std::array<std::reference_wrapper<int>, 3> refs{ std::ref(a), std::ref(b), std::ref(c) };

    for (auto& r : refs)
        r.get() *= 2;          // doubles each original variable

    std::cout << a << ' ' << b << ' ' << c << '\n'; // 2 4 6
}
```

### Accessing the underlying reference

Call `.get()` on a `std::reference_wrapper` to obtain the underlying `T&`. You need `.get()` when the compiler cannot implicitly convert — for example, when calling a member function or using compound assignment. In contexts that expect a `T&` (such as passing to a function taking `int&`), the implicit conversion kicks in automatically.

### std::ref vs std::cref

`std::ref(x)` produces a mutable wrapper — changes through it modify the original. `std::cref(x)` produces a `const` wrapper — the original can be read but not written. Choose `std::cref` when you only need to observe values, following the same const-correctness principle you apply to regular references.

### Key takeaways

- `std::array` cannot hold plain references; use `std::reference_wrapper<T>` instead.
- Create wrappers with `std::ref()` (mutable) or `std::cref()` (read-only).
- Use `.get()` when implicit conversion to `T&` does not apply.
